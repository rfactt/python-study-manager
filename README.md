# Python Study Manager

Gerenciador de estudos via terminal feito em Python com SQLite.

## Sobre o projeto

Este projeto permite cadastrar matérias, criar tarefas de estudo, listar tarefas, atualizar status e deletar registros.

## Funcionalidades

* Cadastrar matérias
* Listar matérias
* Cadastrar tarefas
* Listar tarefas
* Atualizar status da tarefa
* Deletar tarefa
* Salvar dados com SQLite

## Tecnologias

* Python
* SQLite
* Git/GitHub

## Como executar

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta:

```bash
cd python-study-manager
```

Execute o projeto:

```bash
python app.py
```

## Status das tarefas

Os status disponíveis são:

* pendente
* fazendo
* concluido

## Prioridades

As prioridades disponíveis são:
